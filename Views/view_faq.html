FAQ

Voici quelques réponses aux questions qui nous ont été le plus demandées

Pour créer un compte il vous suffit de vous joindre sur la page d'accueil et de cliquer sur le bouton "sign up". Vous serez emener à indiquer vos informations pour créer votre compte.

Pour jouer aux différents jeux présents su rle site, il faut dans un premier temps créer un compte utilisateur. Puis lorsque vous serez sur votre profil utilisateur, il vous suffira de cliquer sur le bouton
"Jouer" du jeux qui vous plait le plus !

Malheureusement, il n'est pas possible d'effectuer une réservation direct sur le site.
Cependant, vous pourrez être rediriger directement sur le site proposant la salle qui vous interesse pour effectuer votre réservation.

Pour les jeux étant proposés sur le site, le but principal est de trouver la bonne énigme.
Comme dans le cadre d'un escape game en physique, votre objectif est de trouver la réponse a 3 enigmes qui vous seront proposé afin de gagner le jeu.

Le niveau des jeux fait référence au niveau de difficulté des 3 enigmes à résoudre.
Trois niveaux sont disponible pour les jeux : Niveau facile, moyen et difficile.

À vous de jouer !
